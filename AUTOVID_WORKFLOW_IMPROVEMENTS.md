# AutoVid 워크플로우 개선 사항

## 현재 문제점
- 영상생성 버튼 → 바로 "빠른서버처리 시작" (에러 발생)
- 단계적 워크플로우 부재

## 필요한 구조 (순서대로)

### 1단계: 대본 생성 및 개선
- 주제 입력 → AI가 짧은 대본 생성
- **대본 미리보기** (수정 가능)
- **AI 개선 버튼** (훅킹, 자연스러움 등)

### 2단계: 프롬프트 설정
- 훅킹 멘트
- 일상/소개 스타일
- 자연스러운 프롬프트 설정
- 각 장면별 정밀 프롬프트 제어

### 3단계: 이미지 생성
- 프롬프트 → 이미지 생성
- 이미지 편집/재생성

### 4단계: TTS 음성 생성
- **목소리 스타일 선택** (이 단계에서)
  - 남성/여성
  - 톤 (밝음, 차분함 등)
  - 속도 조절
- 대본 읽는 TTS 생성 버튼
- 음성 미리보기

### 5단계: 영상 생성
- 이미지 + 음성 + 대본 → 최종 영상 생성
